# Jobs-to-be-Done

JobLane is built around six public-generic Job families:

| Job | Outcome |
|---|---|
| A. Public presence | Publish work with taste gates and draft/live boundaries. |
| B. Keep me strong | In-the-moment coaching and durable body history. |
| C. Trading intelligence | Read-only diagnosis and anomaly surfacing. |
| D. Chief of staff | Morning prioritization, delegation, and saying no. |
| E. Memory/reflection | Remembered reflection with gated durable takeaways. |
| F. Small experiment | Minimal approve/skip workflow with failure alerting. |

The current implementation is a tracer-bullet proof, not a finished product.

## Acceptance Signals

The scorecard does not give credit for a lane name alone. Each Job needs a
typed artifact with a Job-specific acceptance signal:

- A: draft-only public packet with multiple posts, review questions, and no live publish.
- B: in-gym session with structured set parsing, progression checks, and a gated log candidate.
- C: read-only trading synthesis with evidence refs and no trade authority.
- D: morning plan with time blocks, delegate, push, decline, commitments, rationale, and recall.
- E: reflection takeaway with themes, recall, and gated durable-memory promotion.
- F: minimal approve/skip experiment packet with failure awareness.
